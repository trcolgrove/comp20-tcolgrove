It is not possible to request data from a different origin or from your local machine. This is incredibly important, as it prevents Javascript running on malicious websites from having access to the same resources as the Javascript being run in the domain of some other trusted website, which could contain personal information about the user such as contact info or credit card information. With the “same-origin” policy in place, malicious data requests from “westealdata.com” will be denied by “Facebook.com” “citizensbank.com” or any different domain website running Javascript.
